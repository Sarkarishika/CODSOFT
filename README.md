# 💻 Java Mini Projects Collection

This repository contains a collection of beginner-to-intermediate level Java projects given as a task demonstrating core programming concepts like OOP, logic building, and user interaction.

---

## 📂 Projects Included

### 🏦 ATM System
A simple ATM simulation that allows users to:
- Check balance
- Deposit money
- Withdraw money  
Built using object-oriented programming concepts.

---

### 🏦 Bank Account System
A basic banking system with features like:
- Account creation
- Balance management
- Transactions (deposit/withdraw)  
Demonstrates encapsulation and class interaction.

---

### 🎯 Number Guessing Game
A fun console-based game where:
- The system generates a random number
- The user tries to guess it
- Feedback is given (too high / too low)

---

### 🎓 Student Grade Calculator
A program that:
- Takes student marks as input
- Calculates total and average
- Assigns grades based on performance

---

### 👨‍🎓 Student Management (Student Class)
Represents student data with:
- Name, marks, and details
- Basic structure for student record handling

---

## 🛠️ Technologies Used
- Java
- OOP Concepts (Classes, Objects, Encapsulation)
- Basic Input/Output
